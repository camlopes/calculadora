# Calculadora

## Descrição do projeto

Desenvolvimento de uma calculadora utilizando Enhanced Switches para verificar qual foi o operador digitado pelo usuário.

Essa calculadora realiza contas com os seguintes operadores: `+, -, *, /, ^.`
